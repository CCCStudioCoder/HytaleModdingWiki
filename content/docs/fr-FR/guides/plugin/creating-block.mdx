Creating custom blocks requires enabling asset packs and organizing your files properly.
Créer des blocs personnalisés requiert d'activer les packs d'assets (ce qui consitue les mods créés à partir de l'asset editor), et d'organiser correctement et proprement ses fichiers.

## Préparation

1. Activez les asset packs dans `manifest.json` en mettant `IncludesAssetPack` à `true`
2. La structure de vos fichiers doit ressembler à ceci:

<Files>
  <Folder name="resources" defaultOpen>
    <Folder name="Server" defaultOpen>
      <Folder name="Item" defaultOpen>
        <Folder name="Items" />
      </Folder>

      <Folder name="Languages" defaultOpen>
        <Folder name="en-US" />
      </Folder>
    </Folder>

    <Folder name="Common" defaultOpen>
      <Folder name="Icons" />

      <Folder name="Blocks" defaultOpen>
        <Folder name="my_new_block" defaultOpen>
          <File name="model.blockymodel" />
        </Folder>
      </Folder>

      <Folder name="BlockTextures" defaultOpen>
        <File name="texture.png" />
      </Folder>
    </Folder>
  </Folder>
</Files>

3. Ajoutez des traductions dans `Server/Languages/en-US/items.lang`:

```
my_new_block.name = Mon bloc
my_new_block.description = Ma description
```

## Bloc cubique basique

Créez `Server/Item/Items/my_new_block.json`:

```json
{
  "TranslationProperties": {
    "Name": "items.my_new_block.name",
    "Description": "items.my_new_block.description"
  },
  "Id": "My_New_Block",
  "MaxStack": 100,
  "Icon": "Icons/ItemsGenerated/my_new_block.png",
  "Categories": [
    "Blocks.Rocks"
  ],

  "PlayerAnimationsId": "Block",
  "Set": "Rock_Stone",
  "BlockType": {
    "Material": "Solid",
    "DrawType": "Cube",
    "Group": "Stone",
    "Flags": {},
    "Gathering": {
      "Breaking": {
        "GatherType": "Rocks",
        "ItemId": "my_new_block"
      }
    },
    "BlockParticleSetId": "Stone",
    "Textures": [
      {
        "All": "BlockTextures/my_new_block.png"
      }
    ],
    "ParticleColor": "#aeae8c",
    "BlockSoundSetId": "Stone",
    "BlockBreakingDecalId": "Breaking_Decals_Rock"
  },
  "ResourceTypes": [
    {
      "Id": "Rock"
    }
  ]
}
```
Ce genre de fichier peut être créé à partir de l'asset editor des creative tools vanilla.

Soyez sûr que tous les fichiers référencés existent dans leurs dossiers respectifs, et que leurs chemins soient corrects.
