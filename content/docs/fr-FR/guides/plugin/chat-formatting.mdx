![Exemple de style de chat](/assets/guides/chat-example.png)

Le chat utilise le PlayerChatEvent, qui contient le formatteur, la PlayerRef pour l'envoyeur, le contenu, ainsi que les destinataires (en considérant qu'ils constituent la liste des joueurs qui pourront voire le message)
Vous pouvez annuler l'envoie et modifier le contenu et le style à tous moments, que ce soit pour un message envoyé par le serveur, ou encore n'importe quel joueur.

## Styles manuels (Approche Standard)

```java
public class ChatFormatter {

  public static void onPlayerChat(PlayerChatEvent event) {
    PlayerRef sender = event.getSender();
    if(event.getContent().equalsIgnoreCase("poo")) { // Retire les messages inappropriés, puis envoie un avertissement à l'envoyeur de ce message.
      event.setCancelled(true);
      sender.sendMessage(Message.raw("Hey, you cannot say that!").color(Color.RED));
    }

    if(event.getContent().equalsIgnoreCase("you stink")) {
      event.setContent("i stink");
    }

    event.setFormatter((playerRed, message) ->  // stylise les "cool" en rouge, les nomp de joueurs en jaune, et les deux-points en rose
        Message.join(
            Message.raw("[COOL] ").color(Color.RED),
            Message.raw(sender.getUsername()).color(Color.YELLOW),
            Message.raw(" : " + message).color(Color.PINK)
        ));

  }
}
```

L'interface du formatteur est la suivante:

```java
  public interface Formatter {
    @Nonnull
    Message format(@Nonnull PlayerRef playerRef, @Nonnull String message);
  }

```

## TinyMessage - Format de texte extensible (Alternative simple)

![exemple de style de chat](/assets/guides/chat-tinymessage-example.png)

**TinyMessage** est un outils léger, puissant et riche fourni par Hytale pour utiliser des tags simples afin de créer des dégradés, d'utiliser des couleurs hexadécimales, de créer les liens et des styles imbriqués dans les messages du chat. Notez que son système ressemble aux ``MiniMessage`` Minecraft.

### Features

- **Dégradés:** `<gradient:red:blue>Hello</gradient>` ou multi-couleur `<gradient:gold:red:black>...`
- **Couleurs hexadécimales:** `<color:#FF55FF>Couleur personnalisée</color>` ou `<color:red>Couleurs simples (déjà nommées)</color>`
- **Standard Styles:** `<b>Bold</b>`, `<i>Italic</i>`, `<u>Underline</u>`, `<mono>Monospace</mono>`
- **Liens externes clickables:** `<link:https://google.com>Click moi!</link>`
- **Styles imbriqués:** Tags can be nested indefinitely

### Exemples d'utilisation

```java
// Transforme une string en message formaté
Message message = TinyMsg.parse("<gradient:red:blue>Hello World!</gradient>");
player.sendMessage(message);

// Styles multiples
TinyMsg.parse("<b><color:gold>Bold Gold Text</color></b>");

// Lien clickable en dégradé
TinyMsg.parse("<link:https://example.com><gradient:aqua:blue>Click me!</gradient></link>");

// Styles imbriqués complexes
TinyMsg.parse("<b>Bold <i>and italic <color:red>and red</color></i></b>");

// Réinitialise les styles au milieu de la string
TinyMsg.parse("<b>Bold <reset>normal text");
```

```java
public class ChatFormatter {

    private void onPlayerChat(PlayerChatEvent event) {
        PlayerRef sender = event.getSender();
        if (event.getContent().equalsIgnoreCase("poo")) {
            event.setCancelled(true);
            sender.sendMessage(TinyMsg.parse("<red>Hey, you cannot say that!</red>"));
        }

        if (event.getContent().equalsIgnoreCase("you stink")) {
            event.setContent("i stink");
        }

        event.setFormatter((playerRed, message) ->
                TinyMsg.parse("<red>[COOL] </red><yellow>" + sender.getUsername() + "</yellow><pink> : " + message + "</pink>"));
  }
}
```

Téléchargez la dernière version de TinyMessage.jar depuis la page GitHub, et placez-là dans le dossier `mods` du dossier de votre serveur.

### Reference des

#### `TinyMsg.parse(String text)`

Analyse une string avec des tags TinyMsg et renvoie un `Message`.

**Paramètres:**

- `text` - La string à analyser

**Sorties:**

- `Message` - Un `Message` Hytale prêt à être envoyéà des joueurs.

### Tags supportés

| Tag                | Alias                 | Exemple                                 | Description                                            |
| ------------------ | --------------------- | --------------------------------------- | -------------------------------------------------------|
| `<color:X>`        | `<c:X>`, `<colour:X>` | `<color:red>text</color>`               | Change la couleur du texte (hexadécimales on prénommé) |
| `<gradient:X:Y:Z>` | `<grnt:X:Y:Z>`        | `<gradient:red:blue>text</gradient>`    | Créer un dégradé de couleurs                           |
| `<bold>`           | `<b>`                 | `<b>text</b>`                           | Metsle texte en gras                                   |
| `<italic>`         | `<i>`, `<em>`         | `<i>text</i>`                           | Mets le texte en iitalique                             |
| `<underline>`      | `<u>`                 | `<u>text</u>`                           | Souligne le texte                                      |
| `<monospace>`      | `<mono>`              | `<mono>text</mono>`                     | Utilise la police monospace                            |
| `<link:URL>`       | `<url:link>`          | `<link:https://google.com>click</link>` | Créer un lien clickable                                |
| `<reset>`          | `<r>`                 | `<b>bold<reset>normal`                  | Réinitialise  tous les styles                          |

### NCouelurs prénommées

`black`: noir, `dark_blue`: bleu foncé, `dark_green`: vert foncé, `dark_aqua`: bleu marine, `dark_red`: rouge bordeaux, `dark_purple`: mauve foncé, `gold`: doré, `gray`: gris , `dark_gray`: gris foncé, `blue`: bleu, `green`: vert, `aqua`: cyan, `red`: rouge, `light_purple`: violet clair, `yellow`: jaune, `white`: blanc.
